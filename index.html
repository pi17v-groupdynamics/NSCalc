Результирующее основание Результат

  1. Выберите один из вариантов работы калькулятора.
  2. Введите значение которое необходимо перевести.
  3. Укажите систему счисления для введенного значения.
  4. Укажите в какую систему счисления переводить.
  5. Нажмите кнопку "Рассчитать".

Калькулятор перевода чисел имеет одно поле для ввода. В это поле необходимо ввести число которое Вы хотите перевести или математическое выражение, выбрав предварительно соответствующий вариант работы.

После этого следует указать в какой системе счисления число/выражение было введено. Для этого предназначен выпадающий список из традиционных систем счисления и дополнительного поля "Другая СС", по нажатию которого предоставляется возможность ввода неодходимой системы счисления в диапазоне [2 .. 36].

Далее необходимо выбрать в какую систему необходимо перевести введенное значение.

После нажмития кнопки "РАССЧИТАТЬ" результат будет отображен в соответствующем поле.
